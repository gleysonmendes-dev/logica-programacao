# Lógica de Programação

Repositório dedicado ao estudo de lógica de programação, algoritmos e fundamentos,
baseado no curso LogicamentePro.

## Conteúdos estudados
- Lógica proposicional
- Algoritmos e pseudocódigo
- Operadores e tipos de dados
- Estruturas condicionais e de repetição
- Vetores, matrizes e funções

## Objetivo
Construir uma base sólida em lógica de programação para aprender linguagens
como JavaScript, Python ou Java no futuro.
